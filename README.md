##Learning to rank for behavioral targeting

### Dataset
   KDD CUP 2012 track2
####Desciption
Predict the click-through rate of ads given the query and user information.
<a href="http://www.kddcup2012.org/c/kddcup2012-track2">KDD CUP 2012 track2</a>
